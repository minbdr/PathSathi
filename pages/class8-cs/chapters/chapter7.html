Chapter 7: Web Page Designing with HTML

What is HTML?

HTML (HyperText Markup Language) is a standard markup language used to create web pages. It describes the structure and content of a web page using special codes called tags.

Key Features of HTML

  • Platform independent
  • Easy to learn and use
  • Supports multimedia elements
  • Case insensitive
  • Works with all web browsers

Basic Structure of HTML Document

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>My First Heading</h1>
        <p>My first paragraph.</p>
    </body>
</html>

HTML Tags and Elements

What are HTML Tags?

HTML tags are keywords enclosed in angle brackets (< >). Most tags come in pairs:

  • Opening tag: <tagname>
  • Closing tag: </tagname>
  • Content: Text between opening and closing tags

Basic HTML Tags

Document Structure Tags

  • <html>: Root element of HTML page
  • <head>: Contains metadata about the document
  • <title>: Sets the title of the web page
  • <body>: Contains the visible content

Heading Tags

<h1>Largest Heading</h1>
<h2>Second Largest</h2>
<h3>Third Largest</h3>
<h4>Fourth Largest</h4>
<h5>Fifth Largest</h5>
<h6>Smallest Heading</h6>

Paragraph and Text Tags

  • <p>: Defines a paragraph
  • <br>: Line break (single tag)
  • <hr>: Horizontal rule/line (single tag)
  • <b>: Bold text
  • <i>: Italic text
  • <u>: Underlined text

List Tags

<!-- Unordered List -->
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<!-- Ordered List -->
<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

Link and Image Tags

<!-- Link -->
<a href="https://www.example.com">Visit Example.com</a>

<!-- Image -->
<img src="image.jpg" alt="Description of image">

HTML Attributes

Attributes provide additional information about HTML elements:

  • href: Specifies the URL for links
  • src: Specifies the source file for images
  • alt: Alternative text for images
  • width & height: Dimensions of elements
  • title: Tooltip text

Creating Tables in HTML

<table border="1">
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
    </tr>
</table>

HTML Forms

<form>
    <label>Name:</label>
    <input type="text" name="username"><br><br>
    
    <label>Email:</label>
    <input type="email" name="email"><br><br>
    
    <input type="submit" value="Submit">
</form>

Complete Example

<!DOCTYPE html>
<html>
<head>
    <title>My School Website</title>
</head>
<body>
    <h1>Welcome to ABC School</h1>
    <hr>
    
    <h2>About Us</h2>
    <p>We are a <b>leading educational institution</b> 
       committed to excellence in education.</p>
    
    <h2>Subjects Offered</h2>
    <ul>
        <li>Mathematics</li>
        <li>Science</li>
        <li>Computer Science</li>
        <li>English</li>
    </ul>
    
    <h2>Contact Us</h2>
    <p>Visit our website: 
       <a href="http://www.abcschool.com">www.abcschool.com</a>
    </p>
    
    <img src="school.jpg" alt="School Building" width="300">
</body>
</html>

Best Practices for HTML

  • Always use proper HTML structure
  • Include DOCTYPE declaration
  • Use meaningful tag names
  • Always close your tags
  • Use alt attributes for images
  • Keep your code organized and indented
  • Validate your HTML code

✅ Practice Activity

  • Your name as the main heading
  • A paragraph about your hobbies
  • A list of your favorite subjects
  • A link to your favorite website
