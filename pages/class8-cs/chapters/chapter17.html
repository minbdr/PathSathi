Welcome to the complete guide on Operating Systems. In this chapter, we will explore the fundamental concepts of operating systems, their types, and how they manage computer resources.

1. Introduction

An Operating System (OS) is a software that acts as an intermediary between computer hardware and the user. It manages all the hardware and software resources of the computer system. The operating system provides a platform for application programs to run and enables users to interact with the computer.

Without an operating system, a computer is just a collection of hardware components that cannot perform any useful tasks. The OS is the first program that loads when a computer starts, and it continues to run until the computer is shut down.

Functions of an Operating System:

  • Process Management: Controls the execution of programs and allocates CPU time to different processes.
  • Memory Management: Manages the computer's primary memory (RAM) and allocates it to different programs.
  • File Management: Organizes and manages files and folders on storage devices.
  • Device Management: Controls and coordinates the use of hardware devices like printers, keyboard, and mouse.
  • Security: Protects the system from unauthorized access and malicious software.
  • User Interface: Provides a way for users to interact with the computer through GUI or command-line interface.

2. Types of Operating System (OS)

Operating systems can be classified into different types based on their functionality, user interface, and usage. Here are the main types:

a) Single-User, Single-Task OS

This type of OS allows only one user to perform one task at a time. Early computer systems used this type of operating system.

Example: MS-DOS (Microsoft Disk Operating System)

b) Single-User, Multi-Task OS

This OS allows a single user to run multiple programs or tasks simultaneously. Most personal computers today use this type.

Examples: Windows 10, Windows 11, macOS, Ubuntu Linux

c) Multi-User OS

This type allows multiple users to access the computer system simultaneously. Each user can run their own programs independently.

Examples: Unix, Linux, Windows Server

d) Real-Time OS (RTOS)

Real-time operating systems are designed to process data and respond to inputs within a specific time frame. They are used in systems where timing is critical.

Examples: VxWorks, RTLinux (used in aircraft control systems, medical equipment, and industrial robots)

e) Mobile OS

Operating systems designed specifically for mobile devices like smartphones and tablets.

Examples: Android, iOS, Windows Mobile

f) Network OS

These operating systems are designed to manage network resources and provide services to multiple computers connected in a network.

Examples: Novell NetWare, Windows Server

3. Difference between CUI and GUI Operating System

Operating systems can be categorized based on their user interface into two main types: Character User Interface (CUI) and Graphical User Interface (GUI).

CUI (Character User Interface)

A CUI-based operating system uses text-based commands for user interaction. Users must type specific commands to perform tasks. It requires knowledge of command syntax.

Example: MS-DOS, Unix shell, Linux command line

GUI (Graphical User Interface)

A GUI-based operating system uses graphical elements like windows, icons, menus, and buttons. Users interact with the system using a mouse and keyboard in a more intuitive way.

Example: Windows, macOS, Ubuntu with GNOME

4. Microsoft Disk Operating System (MS-DOS)

MS-DOS (Microsoft Disk Operating System) is a command-line based operating system developed by Microsoft in 1981. It was one of the most widely used operating systems in the 1980s and early 1990s before the introduction of graphical user interfaces.

Characteristics of MS-DOS:

  • Character User Interface (CUI): MS-DOS uses text-based commands for all operations.
  • Single-User, Single-Task: It allows only one user to perform one task at a time.
  • 16-bit Operating System: MS-DOS is a 16-bit operating system, which means it can handle 16 bits of data at a time.
  • Command-Line Interface: All tasks are performed by typing specific commands at the command prompt (C:\>).
  • File Management: Provides basic file management operations like creating, copying, deleting, and renaming files.
  • Low Memory Requirements: Requires very little memory compared to modern operating systems.

Features of MS-DOS:

  • Simple and lightweight
  • Direct hardware access
  • Efficient for batch processing
  • Supports various disk formats (floppy disks, hard disks)
  • Foundation for early Windows versions (Windows 3.1, Windows 95/98)

Some Terms Used in DOS:

1. Booting

Booting is the process of loading system files into the computer's memory from disk. When you turn on the computer, the operating system is loaded from the hard disk into the computer's memory (RAM). This process prepares the computer for use.

Types of Booting:

  • Cold Booting: Starting the computer by pressing the power button when it is completely off.
  • Warm Booting: Restarting the computer using Ctrl+Alt+Del keys or the reset button without turning off the power.
2. File

A file is a collection of related data or information stored on a disk with a unique name. Files can contain text, programs, images, or other types of data. In MS-DOS, every file has a name and an extension.

File Naming Convention:

  • File Name: Can be up to 8 characters long (in DOS)
  • Extension: Optional, up to 3 characters, separated by a dot (.)
  • Example: LETTER.TXT, GAME.EXE, PHOTO.JPG
3. Directory

A directory (also called a folder) is a container that stores files and other directories. It helps organize files in a hierarchical structure, making it easier to manage and locate them.

Types of Directories:

  • Root Directory: The main directory of a disk, represented by a backslash (\). Example: C:\
  • Subdirectory: A directory created inside another directory. Example: C:\WINDOWS\SYSTEM
  • Parent Directory: The directory that contains the current directory.
  • Current Directory: The directory you are currently working in.
4. Wildcards

Wildcards are special characters used in DOS commands to represent one or more characters in file names. They help perform operations on multiple files at once.

Types of Wildcards:

  • Asterisk (*): Represents any number of characters (zero or more).
    DIR *.TXT (displays all files with .TXT extension)
    DEL A*.* (deletes all files starting with A)
  • Question Mark (?): Represents a single character.
    DIR FILE?.TXT (displays FILE1.TXT, FILE2.TXT, etc.)
    COPY ???.DOC A: (copies all 3-letter named .DOC files)
5. Command

A command is an instruction given to the computer to perform a specific task. In DOS, commands are typed at the command prompt (C:\>) and executed by pressing the Enter key.

Structure of a DOS Command:

  • Command Name: The name of the command (e.g., DIR, COPY, DEL)
  • Parameters: Additional information needed by the command (e.g., file names, drive letters)
  • Switches: Optional modifiers that change how the command works (e.g., /P, /W, /S)

Example:

DIR /P (DIR is the command, /P is a switch for page-by-page display)

5. MS-DOS Commands

MS-DOS commands are instructions that users type at the command prompt to perform various tasks. These commands are categorized into two types:

i) Internal Commands

Internal commands (also called resident commands) are built into the operating system and are loaded into memory when the computer starts. These commands are always available and do not require any external files to execute.

Common Internal Commands:

  • DIR: Displays a list of files and subdirectories in a directory.
    C:\>DIR
  • CD or CHDIR: Changes the current directory.
    C:\>CD WINDOWS
  • MD or MKDIR: Creates a new directory.
    C:\>MD NEWFOLDER
  • RD or RMDIR: Removes/deletes an empty directory.
    C:\>RD OLDFOLDER
  • COPY: Copies one or more files from one location to another.
    C:\>COPY FILE1.TXT D:\BACKUP
  • DEL or ERASE: Deletes one or more files.
    C:\>DEL FILE1.TXT
  • REN or RENAME: Renames a file or directory.
    C:\>REN OLDNAME.TXT NEWNAME.TXT
  • TYPE: Displays the contents of a text file.
    C:\>TYPE FILE.TXT
  • CLS: Clears the screen.
    C:\>CLS
  • DATE: Displays or sets the system date.
    C:\>DATE
  • TIME: Displays or sets the system time.
    C:\>TIME
  • VER: Displays the version of MS-DOS.
    C:\>VER
  • VOL: Displays the disk volume label and serial number.
    C:\>VOL

ii) External Commands

External commands (also called transient commands) are not built into the operating system. They are stored as separate executable files (.COM or .EXE) in the DOS directory. These commands are loaded into memory only when they are executed.

Common External Commands:

  • FORMAT: Formats a disk to prepare it for storing data.
    C:\>FORMAT A:
  • XCOPY: Copies files and directory trees to another location.
    C:\>XCOPY C:\DOCS D:\BACKUP /S
  • DISKCOPY: Copies the entire contents of one floppy disk to another.
    C:\>DISKCOPY A: B:
  • CHKDSK: Checks the disk for errors and displays a status report.
    C:\>CHKDSK C:
  • LABEL: Creates, changes, or deletes the volume label of a disk.
    C:\>LABEL C: MYDISK
  • ATTRIB: Displays or changes file attributes (read-only, hidden, system, archive).
    C:\>ATTRIB +R FILE.TXT
  • TREE: Displays the directory structure of a drive or path graphically.
    C:\>TREE C:\
  • MOVE: Moves files from one directory to another.
    C:\>MOVE FILE.TXT D:\FOLDER
  • BACKUP: Backs up files from one disk to another.
    C:\>BACKUP C:\*.* A: /S
  • RESTORE: Restores files that were backed up using the BACKUP command.
    C:\>RESTORE A: C:\*.* /S
Aspect Internal Commands External Commands
Location Built into COMMAND.COM Stored as separate files (.COM/.EXE)
Memory Always loaded in memory Loaded only when executed
Speed Faster execution Slower execution
File Required No separate file needed Requires external file to execute
Examples DIR, CD, COPY, DEL, CLS, DATE, TIME FORMAT, XCOPY, CHKDSK, ATTRIB, TREE

6. MS Windows

Microsoft Windows (commonly known as Windows) is a series of graphical operating systems developed by Microsoft. It is the most widely used operating system for personal computers worldwide. Windows provides a user-friendly GUI (Graphical User Interface) that makes it easy for users to interact with the computer.

History of Windows:

  • Windows 1.0 (1985): The first version of Windows, introduced as a graphical extension to MS-DOS.
  • Windows 3.1 (1992): Gained popularity with improved interface and functionality.
  • Windows 95 (1995): Introduced the Start Menu, taskbar, and revolutionary user interface.
  • Windows 98 (1998): Enhanced with better hardware support and Internet integration.
  • Windows XP (2001): One of the most successful versions with improved stability and performance.
  • Windows Vista (2007): Introduced Aero interface with visual effects.
  • Windows 7 (2009): Highly popular, with improved performance and user experience.
  • Windows 8 (2012): Introduced touch-friendly Metro interface.
  • Windows 10 (2015): Unified interface for all devices with regular updates.
  • Windows 11 (2021): Latest version with modern design and enhanced features.

Features of MS Windows:

  • Graphical User Interface (GUI): Easy-to-use visual interface with windows, icons, and menus.
  • Multitasking: Ability to run multiple programs simultaneously.
  • Plug and Play: Automatic detection and configuration of hardware devices.
  • File Management: Windows Explorer/File Explorer for managing files and folders.
  • Networking: Built-in networking capabilities for internet and local network access.
  • Security: Windows Defender, firewall, and user account control for system protection.
  • Multimedia Support: Built-in support for audio, video, and graphics.
  • Application Support: Compatible with millions of software applications.
  • Regular Updates: Continuous security and feature updates from Microsoft.
  • User-Friendly: Intuitive design suitable for all types of users.

Components of Windows Desktop:

  • Desktop: The main screen where icons, files, and folders are displayed.
  • Start Menu: Provides access to programs, settings, and power options.
  • Taskbar: Shows running programs and quick access icons.
  • System Tray: Displays system icons like volume, network, and date/time.
  • Icons: Graphical representations of programs, files, or folders.
  • Windows: Rectangular frames that display application content.
  • File Explorer: Tool for browsing and managing files and folders.
  • Recycle Bin: Temporary storage for deleted files.

Advantages of MS Windows:

  • User-friendly and easy to learn
  • Wide hardware and software compatibility
  • Excellent for gaming and multimedia
  • Strong customer support and documentation
  • Regular security updates
  • Suitable for both personal and professional use

Disadvantages of MS Windows:

  • Requires license purchase (not free)
  • More vulnerable to viruses and malware compared to other OS
  • Can be resource-intensive (requires more RAM and storage)
  • May experience occasional system slowdowns
  • Forced automatic updates can be disruptive

üí° Key Takeaways

  • Operating systems manage hardware and software resources of a computer.
  • Different types of OS serve different purposes: single-user, multi-user, real-time, mobile, and network OS.
  • CUI uses text commands while GUI uses graphical elements for user interaction.
  • MS-DOS is a command-line operating system that was popular in the 1980s and 1990s.
  • DOS commands are categorized into Internal (built-in) and External (separate files) commands.
  • MS Windows is a graphical operating system that provides a user-friendly interface for all types of users.

Questions and Answers

Q1. What is operating system? Write down the advantage of Windows based OS over DOS based OS.

Answer:

What is Operating System:

An operating system (OS) is system software that manages computer hardware and software resources and provides common services for computer programs. It acts as an intermediary between users and the computer hardware, making it possible for users to interact with the computer without needing to understand the complex details of hardware operations. The operating system is the first program loaded when a computer starts and continues running until the computer is shut down.

Advantages of Windows-Based OS over DOS-Based OS:

  • Windows uses GUI (icons & mouse), DOS uses text commands: Windows provides a graphical user interface with visual elements like icons, windows, and menus that can be operated with a mouse, making it much more intuitive. DOS requires typing specific text commands, which is less user-friendly.
  • Easier to learn and use: Windows has a simple point-and-click interface that anyone can learn quickly without technical knowledge. DOS requires memorizing commands and their syntax, making it difficult for beginners.
  • Supports multitasking (run many programs at once): Windows allows users to run multiple programs simultaneously and switch between them easily. DOS is single-tasking and can only run one program at a time.
  • Better graphics and sound support: Windows has excellent built-in support for multimedia, including high-quality graphics, audio, video playback, and gaming. DOS has very limited or no multimedia capabilities.
  • Supports modern hardware and software: Windows supports the latest hardware devices with Plug and Play functionality and is compatible with millions of modern software applications. DOS has limited hardware support and most modern software does not work on DOS.
  • Provides file management tools and networking: Windows includes File Explorer for easy visual file management with drag-and-drop functionality, and built-in networking features for internet access and file sharing. DOS requires command-line operations for file management and has limited networking capabilities.
