Chapter 3: Spreadsheet Software

Introduction to Spreadsheet

A spreadsheet is a computer application that simulates a paper worksheet. It displays multiple cells that form a grid of rows and columns, with each cell capable of containing either text, numbers, or formulas.

Key Topics Covered

  • Introduction to spreadsheet applications (MS Excel, LibreOffice Calc)
  • Understanding cells, rows, and columns
  • Data entry and editing
  • Basic formulas and functions
  • Formatting cells and data
  • Creating charts and graphs
  • Sorting and filtering data
