Chapter 13: Introduction to QBasic

What is QBasic?

QBasic (Quick Beginner's All-purpose Symbolic Instruction Code) is a high-level programming language developed by Microsoft. It's an ideal language for beginners to learn programming concepts.

Features of QBasic

  • Easy to learn and understand
  • English-like syntax
  • Interactive environment
  • Built-in editor and debugger
  • Immediate execution of commands
  • Good for beginners

QBasic Environment

The QBasic environment consists of:

  • Menu Bar: Contains File, Edit, View, Search, Run, Debug, Options, Help
  • Editor Window: Where you write your programs
  • Immediate Window: For testing commands directly
  • Output Screen: Shows program results

Basic QBasic Commands

1. PRINT Statement

Used to display output on the screen.

PRINT "Hello World"
PRINT "My name is John"
PRINT 5 + 3

2. CLS Statement

Clears the screen.

CLS

3. REM Statement

Used to add comments in the program.

REM This is a comment
' This is also a comment

Variables in QBasic

Variables are used to store data. QBasic supports different types of variables:

Variable Types

  • Numeric Variables: Store numbers (A, B, NUM, X1)
  • String Variables: Store text (NAME$, ADDRESS$)

Rules for Variable Names

  • Must start with a letter
  • Can contain letters, numbers, and periods
  • Cannot exceed 40 characters
  • Cannot be a reserved word
  • String variables must end with $ sign

INPUT Statement

Used to accept input from the user.

INPUT "Enter your name: "; NAME$
INPUT "Enter your age: "; AGE
PRINT "Hello "; NAME$
PRINT "You are "; AGE; " years old"

Arithmetic Operators

Operator Operation Example
+ Addition 5 + 3 = 8
- Subtraction 5 - 3 = 2
* Multiplication 5 * 3 = 15
/ Division 6 / 3 = 2
^ Exponentiation 2 ^ 3 = 8

Sample Programs

Program 1: Addition of Two Numbers

CLS
INPUT "Enter first number: "; A
INPUT "Enter second number: "; B
SUM = A + B
PRINT "The sum is: "; SUM
END

Program 2: Area of Rectangle

CLS
INPUT "Enter length: "; LENGTH
INPUT "Enter width: "; WIDTH
AREA = LENGTH * WIDTH
PRINT "Area of rectangle is: "; AREA
END

Program 3: Simple Calculator

CLS
INPUT "Enter first number: "; NUM1
INPUT "Enter second number: "; NUM2
PRINT "Addition: "; NUM1 + NUM2
PRINT "Subtraction: "; NUM1 - NUM2
PRINT "Multiplication: "; NUM1 * NUM2
PRINT "Division: "; NUM1 / NUM2
END

Built-in Functions

  • ABS(x): Returns absolute value
  • SQR(x): Returns square root
  • INT(x): Returns integer part
  • RND: Returns random number between 0 and 1
  • LEN(string$): Returns length of string

Running a QBasic Program

  1. Type your program in the editor
  2. Press F5 or go to Run menu and select Start
  3. The program will execute and show output
  4. Press any key to return to the editor

  2. Create a program to convert temperature from Celsius to Fahrenheit.
  3. Write a program to calculate simple interest.
  4. Create a program to find the average of three numbers.
